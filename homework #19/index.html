OK... What?

  • Is a build tool for JavaScript.
  • Allows using Node.js NPM packages on client side in a browser.
  • Automizes work
  • Browserify is perhaps the simplest bundler because it uses NodeJS require syntax to import javascript.
  • It does not deal with any other assets other than pure javascript modules.

Prerequisites

  • Node.js
  • NPM
  • Browserify
  • JavaScript modules you will use for development

Getting started

First you need to install Node.js and npm. For example via homepage download or Homebrew.

Then install Browserify via npm globally.
npm install -g browserify

Import and Export

All installed npm packages are saved to node_modules directory and can be imported with standard ES6 syntax. const module = require('module_name');

Our custom code can be exported as a ES6 module.

module.exports = { functionName };

Running Browserify

When we run Browserify, itâ€™ll want to build a new JavaScript file with all of our attached modules (npm modules included). Running this command from project root would look like:

What with exported files

You can use them on your webpage: <script src="js/output.js"> </script>.

Browserify accepts also imports of our ES6 files in ES6 format. Those have to be exported as a ES6 module.

package.json

Browserify is a DevDependency in a project.

In case we export project with Browserify and someone else reuse our code, then it's not necessary install all dependencies we used for development.

Those are already included in our exported file. :-)

Automation with package.json

It's handy to define scripts in package.json: 

"scripts": {
    "build-js": "browserify js/main.js >
     js/findem.js",
    "watch-js": "watchify js/main.js -o
     js/findem.js"
    }

Then just run npm command: npm run build-js or npm run watch-js.

DevDependency Watchify re-run Browserify each time, when exported JS file changes.

Generating source maps in npm

We can set commands to package.json to generate source maps with Browserify and Watchify. 

"scripts": {
    "build-js": "browserify js/main.js > 
        js/findem.js -d",
    "watch-js": "watchify js/main.js -o 
        js/findem.js -d"
}

or just combine them: "watch-js": "watchify js/main.js -o js/findem.js -dv"

Grunt

Together with Browserify we can automate tasks by using Grunt. Grunt requires install 3 separate dependencies and set up a GruntFile.js, where we define our tasks to run.

Gulp

Thanks to Browserify streaming API we can also use Gulp, which is aside of Grunt other option.

Gulp requires install 2 separate dependencies and set up a GulpFile.js.
